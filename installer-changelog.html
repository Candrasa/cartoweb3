CartoWeb3 Installer ChangeLog

Version 3.3.0

(2006-08-31)

  • Includes MapServer 4.8.3 (ChangeLog).

Version 3.2.0b

(2006-02-27)

  • Includes MapServer 4.8.1 (ChangeLog).
  • Includes MapServer, OGR, Proj4, GDAL binaries.
  • Includes OWTChart, a CGI library for charts drawing.
  • Installation of WAMP directly in the C:\ directory is no more mandatory.
  • Installation/uninstallation of WAMP (not included in the installer), CartoWeb and its demos are now fully independent and complete (no file remaining when uninstalling).
  • Proj4 is now installed within the WAMP directory (previously in C:\).

NOTE CartoWeb version (3.2.0) has not changed between CartoWeb Installer 3.2.0 and CartoWeb Installer 3.2.0b.
