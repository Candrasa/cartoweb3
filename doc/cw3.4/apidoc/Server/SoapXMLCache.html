		void printSoapXML(
string
$soapRequest)
    

    


	
		Prints SOAP XML, cache if necessary

Possible cases: 
  • Cache not used --> reads from server
  • Still not cached, first call --> reads from server
  • Still not cached, second call --> reads from server, writes cache
  • Cached --> reads from cache
